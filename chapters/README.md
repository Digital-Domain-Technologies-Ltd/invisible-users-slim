# MX-Don't Make the AI Think - Chapters

This is a **content-only repository** containing markdown chapters for the "MX-Don't Make the AI Think" book.

## Important Notes

- **No build tooling required** - This repository contains only markdown content
- **No npm dependencies** - No `package.json`, no `npm install` needed
- **Build from parent repository** - All PDF generation, linting, and build commands run from the parent `invisible-users` repository

## Content Structure

- `0-cover.md` - Cover page and metadata
- `metadata.yaml` - Pandoc metadata for PDF generation
- `illustrations/` - SVG source files and generated PNG illustrations

## Usage

This repository is designed to be used as a git submodule within the main `invisible-users` project. Standalone usage is not supported.

For complete documentation, build commands, and usage instructions, see the parent repository:

- Main repository: [invisible-users](https://github.com/ddttom/invisible-users)
- Parent README: [../../README.md](../../README.md)
- Git workflow: [../../docs/architecture/GIT-README.md](../../docs/architecture/GIT-README.md)
